## 📖 Cara Kerja Skills

### ❓ Apakah Skills Dijalankan Otomatis?

**YA dan TIDAK** - tergantung konteks:

#### ✅ **Otomatis (Tanpa Perintah Eksplisit)**
Skills akan **otomatis digunakan** saat saya mendeteksi tugas yang relevan:

**Contoh:**
- Anda bilang: "Ada bug di checkout" → Saya otomatis pakai skill `systematic-debugging`
- Anda bilang: "Buatkan test untuk CartController" → Saya otomatis pakai skill `test-driven-development`
- Anda bilang: "Review code ini" → Saya otomatis pakai skill `code-review-excellence`
- Anda bilang: "Design database untuk notifications" → Saya otomatis pakai skill `database-schema-designer`

#### ❌ **Tidak Otomatis (Perlu Konteks)**
Skills **TIDAK akan dijalankan** jika:
- Tidak ada tugas yang diberikan
- Tugas tidak relevan dengan skill yang ada
- Anda hanya bertanya tentang skills (seperti sekarang)

### 🎯 Kapan Skills Aktif?
Skills aktif saat Anda memberikan **tugas konkret**, seperti:
- "Fix bug X"
- "Buatkan fitur Y"
- "Optimize query Z"
- "Review code A"
- "Design UI untuk B"
