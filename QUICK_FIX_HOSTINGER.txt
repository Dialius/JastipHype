🔧 QUICK FIX - HOSTINGER AUTO-DEPLOY ERROR

MASALAH:
- Hostinger mencoba build npm sendiri
- Error: "Failed to install dependencies"
- Bentrok dengan GitHub Actions

SOLUSI CEPAT:
1. Login hPanel: https://hpanel.hostinger.com
2. Websites → Manage (jastiphype.shop)
3. Cari section "Git" atau "Deployments"
4. MATIKAN "Auto Deploy" toggle
5. HAPUS "Build command" (kalau ada)
6. Save changes

SETELAH ITU:
- Push ke GitHub → GitHub Actions jalan ✅
- Hostinger tidak build lagi ✅
- Deployment berhasil ✅
